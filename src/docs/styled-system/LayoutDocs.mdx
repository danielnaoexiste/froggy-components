#### Layout Props

| Prop          | CSS Property   | Value            |
| :------------ | :------------- | :--------------- |
| width         | width          | string or number |
| height        | height         | string or number |
| minWidth      | min-width      | string or number |
| maxWidth      | max-width      | string or number |
| minHeight     | min-height     | string or number |
| maxHeight     | max-height     | string or number |
| size          | width, height  | string or number |
| display       | display        | string           |
| verticalAlign | vertical-align | string           |
| overflow      | overflow       | string           |
| overflowX     | overflowX      | string           |
| overflowY     | overflowY      | string           |
