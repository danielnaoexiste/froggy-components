#### Position Props

| Prop     | CSS Property | Value            |
| :------- | :----------- | :--------------- |
| position | position     | string           |
| zIndex   | z-index      | string or number |
| top      | top          | string           |
| right    | right        | string           |
| bottom   | bottom       | string           |
| left     | left         | string           |
