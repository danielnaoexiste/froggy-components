#### Typography Props

| Prop          | CSS Property   | Value            |
| :------------ | :------------- | :--------------- |
| fontFamily    | font-family    | string           |
| fontSize      | font-size      | string or number |
| fontWeight    | font-weight    | string or number |
| lineHeight    | line-height    | string or number |
| letterSpacing | letter-spacing | string or number |
| textAlign     | text-align     | string           |
| fontStyle     | font-style     | string           |
