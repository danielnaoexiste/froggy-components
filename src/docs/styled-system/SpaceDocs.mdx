#### Space Props

| Prop              | CSS Property                   | Value            |
| :---------------- | :----------------------------- | :--------------- |
| m, margin         | margin                         | string or number |
| mt, marginTop     | margin-top                     | string or number |
| mr, marginRight   | margin-right                   | string or number |
| mb, marginBottom  | margin-bottom                  | string or number |
| ml, marginLeft    | margin-left                    | string or number |
| mx, marginX       | margin-left and margin-right   | string or number |
| my, marginY       | margin-top and margin-bottom   | string or number |
| p, padding        | padding                        | string or number |
| pt, paddingTop    | padding-top                    | string or number |
| pr, paddingRight  | padding-right                  | string or number |
| pb, paddingBottom | padding-bottom                 | string or number |
| pl, paddingLeft   | padding-left                   | string or number |
| px, paddingX      | padding-left and padding-right | string or number |
| py, paddingY      | padding-top and padding-bottom | string or number |
